## RSS 阅读器

### 说明

本项目的数据来源于[RSShub](https://github.com/DIYgod/RSSHub)，界面展示采用[AdminLTE](https://adminlte.io)。

项目1.0阶段采用传统的HTML页面搭建，数据都保存在Web SQL中。

当前使用界面比较丑，仅仅做了订阅数据的展示，尚未优化前端展示样式。

**测试可用浏览器** [建议升至最新版]

- Chrome 
- ~~Safari~~
- ~~Firefox~~
- ~~IE~~
