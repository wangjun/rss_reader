## RSS 阅读器

### 说明

本项目的数据来源于[RSShub](https://github.com/DIYgod/RSSHub)，界面展示采用[AdminLTE](https://adminlte.io/themes/AdminLTE/index2.html)。

项目1.0阶段采用传统的HTML页面搭建，数据都保存在Web SQL中。

目前处在开发阶段，尚未正式使用。
